# Me RPG — Настольная ролевая система

**Me RPG** — упрощённая и динамичная настольная ролевая система в духе классического фэнтези. Создана как альтернатива D&D 5e с акцентом на быстрые бои, массовые столкновения и гибкую прокачку персонажей.

## Особенности системы

- **Динамичные бои** — быстрые, тактические, с поддержкой массовых сражений
- **Ветки заклинаний** — изучение магии по цепочкам (например: Огненная стрела → Огненный шар → Метеоритный дождь)
- **Ветки навыков и талантов** — прогрессия через предварительные умения
- **Размытые классы** — только стартовое направление, дальнейшее развитие свободное
- **Классические расы** — люди, эльфы, дварфы, полурослики, орки и другие
