# Справочник

Краткий справочник по терминам, сокращениям и основным формулам системы Me RPG.

---

## Термины и сокращения

### Атрибуты

| Сокращение | Полное название | Описание |
|------------|-----------------|----------|
| **СИЛ** | Сила | Физическая мощь, урон в ближнем бою |
| **ЛОВ** | Ловкость | Координация, инициатива, уклонение |
| **ВЫН** | Выносливость | Здоровье, сопротивление урону |
| **ДУХ** | Дух | Магия, воля, восприятие |

### Ресурсы и характеристики

| Сокращение | Полное название | Описание |
|------------|-----------------|----------|
| **ОЗ** | Очки здоровья | Запас здоровья персонажа или существа |
| **ЗЩ** | Защита | Сложность попадания по цели (броня + уклонение) |
| **ОМ** | Очки маны | Ресурс для заклинаний |
| **ОВ** | Очки выносливости | Ресурс для специальных приёмов в бою |
| **XP** | Опыт | Очки опыта для повышения уровня |

### Бой и проверки

| Термин | Описание |
|--------|----------|
| **СЛ** | Сложность — целевое значение для успешной проверки |
| **3d6** | Три шестигранных кубика — базовый бросок системы |
| **Кость истории** | Выделенный кубик (другой цвет/размер), влияет при дубле на двух обычных |
| **Раунд** | 10 секунд боя; каждый участник выполняет действие, перемещение и бонусное действие |
| **Зона** | Участок поля боя ~10×10 м; в массовых боях — одна зона на отряд + до 3 героев |
| **Минион** | Слабое существо с 1 ОЗ, погибает от одного удара |
| **Элита** | Усиленное существо: полные ОЗ, +2 к атаке и урону |
| **Босс** | Мощный противник: ОЗ × 2, два действия за раунд, фазы при 50% и 25% ОЗ |
| **Отряд** | Группа существ в массовом бою (малая 5, средняя 15, большая 30, огромная 50+) |
| **Opportunity** | Атака возможности — провоцируется при определённых действиях |
| **Рана** | Механический эффект от урона: лёгкая (−1), средняя (−2), тяжёлая (−2/−2 и половина скорости) |
| **Лёгкая рана** | Крит или урон ≥ ¼ макс. ОЗ; −1 к атаке, ЗЩ или проверкам |
| **Средняя рана** | Урон ≥ ½ макс. ОЗ; −2 к атаке или ЗЩ |
| **Тяжёлая рана** | 0 ОЗ или урон ≥ макс. ОЗ; −2 к атаке и ЗЩ, половина скорости |

### Магия

| Термин | Описание |
|--------|----------|
| **Концентрация** | Поддержание активного заклинания; при уроне — проверка ДУХ или сброс |
| **Ритуал** | Заклинание без затрат ОМ, требует 10+ минут |
| **Корневое заклинание** | Первое в ветке, изучается без требований |
| **Ветка заклинаний** | Древовидная цепочка заклинаний (родитель → потомок) |

### Развитие персонажа

| Термин | Описание |
|--------|----------|
| **Направление** | Стартовая специализация (воин, стрелок, маг, жрец, плут, бард) |
| **Ранг навыка** | Уровень владения навыком (1–3); бонус +2 за ранг |
| **Талант** | Мощная способность, требует цепочки навыков |
| **Владение** | Навык владения оружием (мечи, копья, луки, кинжалы) |

### Снаряжение

| Термин | Описание |
|--------|----------|
| **Сопротивление урону** | Снижает входящий урон на указанное значение (минимум 1) |
| **Уязвимость** | Увеличение получаемого урона (×1.5) |
| **Преимущество** | Бросок двух кубиков, берётся лучший результат |
| **Помеха** | Бросок двух кубиков, берётся худший результат |

## Основные формулы

### Атрибуты и модификаторы

**Модификатор атрибута:**
```
модификатор = (значение − 10) ÷ 2, округление вниз
```

| Значение | Модификатор |
|----------|-------------|
| 8 | −1 |
| 10 | 0 |
| 12 | +1 |
| 14 | +2 |
| 16 | +3 |
| 18 | +4 |

---

### Проверки (броски)

**Базовая проверка:**
```
3d6 + модификатор атрибута + бонус навыка/таланта ≥ СЛ
```

**Стандартные сложности (СЛ):**

| СЛ | Описание |
|----|----------|
| 8 | Очень легко |
| 10 | Легко |
| 12 | Обычно |
| 14 | Сложно |
| 16 | Очень сложно |
| 18 | Почти невозможно |

**Особые результаты:**
- **Критический успех:** сумма 17–18 на 3d6
- **Критический провал:** сумма 3–4 на 3d6

---

### Здоровье и защита

**Очки здоровья (ОЗ):**
```
ОЗ = ВЫН + (уровень × 3)
```

**Защита (ЗЩ):**
```
ЗЩ = 10 + модификатор ЛОВ + бонус брони
```

| Броня | Бонус ЗЩ | Штраф |
|-------|----------|-------|
| Нет | 0 | — |
| Лёгкая | +2 | — |
| Средняя | +4 | −1 к ЛОВ-проверкам |
| Тяжёлая | +6 | −2 к ЛОВ-проверкам |

---

### Бой

**Инициатива:**
```
3d6 + модификатор ЛОВ
```

**Проверка попадания при атаке:**
```
3d6 + модификатор СИЛ (ближний) или ЛОВ (дальний) + бонус владения ≥ ЗЩ цели
```

**Урон:**
```
Базовый урон оружия + модификатор СИЛ (ближний) или ЛОВ (дальний)
```

**Критический удар:** сумма 17–18 → урон удваивается (бросить кубики дважды)

---

### Магия

**Очки маны (ОМ):**
```
ОМ = ДУХ × 2 + уровень × 3
```

**Восстановление:** все ОМ за длительный отдых, половина за короткий.

**Проверка концентрации при уроне:**
```
СЛ = половина полученного урона
```

---

### Ресурсы в бою

**Очки выносливости (ОВ):**
```
Максимум ОВ = уровень + модификатор ВЫН
```

**Восстановление:** 1 ОВ за короткий отдых, все за длительный.

|---------|--------------|
| 1 | 0 |
| 2 | 100 |
| 3 | 250 |
| 4 | 500 |
| 5 | 900 |
| 6 | 1400 |
| 7 | 2000 |
| 8 | 2800 |
| 9 | 3800 |
| 10 | 5000 |

**За повышение уровня:** +1 к одному атрибуту, очки навыков/талантов, очки заклинаний (если применимо).

### Массовые бои — отряды

| Размер отряда | ОЗ отряда | Урон отряда | ЗЩ |
|---------------|-----------|-------------|-----|
| Малая (5) | 25 | 2d6 | 12 |
| Средняя (15) | 75 | 4d6 | 12 |
| Большая (30) | 150 | 6d6 | 11 |
| Огромная (50+) | 250 | 8d6 | 10 |

**Механика:** каждые 25 потерянных ОЗ отряда — −1 к его урону. При 0 ОЗ отряд рассеян.
