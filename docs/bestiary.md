# Бестиарий

Базовые существа для встреч. Значения даны для одиночных противников; для массовых боёв используйте правила отрядов из [Системы боя](combat-system).

---

## Легенда

- **ОЗ** — очки здоровья
- **ЗЩ** — защита (сложность попадания)
- **Атака** — бонус к броску атаки
- **Урон** — базовый урон
- **Скорость** — метров за раунд
